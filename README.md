-----------------------------------------------------
**WHAT YOU NEED TO MAKE OTNEZ WORK! :**
-----------------------------------------------------

Source code (zip) extracted in one folder!

-create .env file and put these keys:
DISCORD_TOKEN=your_token
GROQ_API=your_api_key
PANIC_API_GROQ=your_panic_key (if wanted)

put .env file in **ROOT** directory! (same as main.py and webserver.py)
when launching the bot, terminal 'python webserver.py' or put 'python webserver.py' as starting command (on Render)
leave the script running for as long as you want otnez to execute


-----------------------------------------------------
**WANT TO TWEAK OTNEZ?**
-----------------------------------------------------

If you want otnez to have a special personality, modify the system_prompt.txt file, it contains the main prompt sent to the AI!
